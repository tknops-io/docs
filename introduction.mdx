# Welcome to tknops

tknops is an AI token cost management platform that helps you monitor, analyze, and optimize your AI token usage and costs.

## What is tknops?

tknops provides comprehensive tools to track and manage your AI token consumption across different models and providers. With tknops, you can:

- **Automatic Usage Tracking**: Capture token usage, cost, and latency with minimal code
- **Privacy-First**: Prompt and response content are NOT stored by default
- **Cost Calculation**: Built-in pricing registry for popular models (OpenAI, Anthropic, and more)
- **Environment Tagging**: Separate analytics for `prod`, `dev`, and `staging` environments
- **Framework Support**: Works seamlessly with OpenAI and LangChain response objects
