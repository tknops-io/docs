# Installation

The **tknOps** SDK is available as a Python package. You can install it using `pip`.

## Prerequisites

- Python 3.7 or higher
- An active tknOps account (to get your API Key)

## Install via pip

Run the following command in your terminal:

```bash
pip install tknops-llm
```

## Verify Installation

You can verify that the package is installed correctly by running:

```bash
pip show tknops-llm
```

This should display the package details, including the version number.
